Общие замечания

На этой персональной странице расположены некоторые программы, написанные 
автором для операционных систем MS-DOS,Microsoft Windows, FreeBSD, Linux и некоторых
других.
Все программы распространяются под GNU GENERAL PUBLIC LICENSE 
или GNU LIBRARY GENERAL PUBLIC LICENSE если условия распространения пропущены.
Можно посылать сообщения на dstef@mail.ru, dmstef@yandex.ru, dmstef@gmail.com, 
dmstef@rambler.ru  для вопросов, советов и возможной помощи по этим программам.
Пожалуйста, не ожидайте помощи по всем программам, особенно, для старого ПО под MS-DOS.
Контрольные суммы для всех архивов можно найти в MD5SUMS и SHA1SUMS.
Конечно, все программы предлагаюся на хорошо известной основе AS IS.
